## Installation
1. Clone the repository
2. Run `./gradlew genSources`
3. Run `./gradlew build`
4. Client jar is in `build/libs/client-base-???.jar` (not `..-sources.jar`)

## Gui Binding
You must use `.bind gui [key]`
